# Guo Qin Project Page

This is a lightweight static academic homepage designed for GitHub Pages.

## Preview locally

Open `index.html` directly in a browser, or run a tiny local server:

```bash
python3 -m http.server 8000
```

Then visit `http://localhost:8000`.

## Publish with GitHub Pages

1. Create a repository named `guoqincode.github.io` on GitHub.
2. Upload `index.html`, `styles.css`, and this `README.md`.
3. Go to `Settings -> Pages`.
4. Set source to `Deploy from a branch`.
5. Choose branch `main` and folder `/root`.
6. Your site will be available at `https://guoqincode.github.io/`.

To publish as a project page instead, create any repository name, upload the files,
enable Pages in the same settings screen, and GitHub will provide a URL like:
`https://guoqincode.github.io/repository-name/`.
